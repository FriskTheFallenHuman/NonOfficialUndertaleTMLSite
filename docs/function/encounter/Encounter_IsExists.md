# Encounter_IsExists {docsify-ignore-all}

## Description
Check if a encounter exist.

## Syntax
```gml
Encounter_IsExists( encounter_id );
```

## Arguments
| Argument | Type | Description |
| :-- | :-- | :-- |
| encounter_id | Integral | The encounter id that you want to check. |

## Returns
Integral
